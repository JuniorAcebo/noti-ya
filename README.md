# 📱 Aplicación de Noticias Sudamericanas

## Descripción del Proyecto

Aplicación móvil (APK) de noticias enfocada en la región sudamericana. Muestra noticias actualizadas en tiempo real mediante el consumo de una API externa. Además, cuenta con funcionalidades sociales y de personalización almacenadas localmente y en un servidor propio.

## Funcionalidades

- Consulta de noticias en tiempo real desde internet
- Cobertura de múltiples países sudamericanos
- Guardado de noticias en **favoritos** de forma local en el dispositivo
- Sistema de **likes y dislikes** por noticia
- Interfaz simple e intuitiva
- Manejo eficiente de datos en formato JSON

## Tecnologías Utilizadas

| Tecnología | Uso |
|---|---|
| Java | Lenguaje principal de desarrollo |
| Android Studio | Entorno de desarrollo (IDE) |
| XML | Diseño de interfaces de usuario |
| Retrofit | Conexión con APIs REST |
| OkHttp | Gestión de peticiones HTTP |
| Gson | Conversión de JSON a objetos Java |
| NewsData.io | Fuente de noticias en tiempo real |
| PythonAnywhere | Hosting de la API propia para likes/dislikes |

## Arquitectura de APIs

- **NewsData.io** — API externa para obtener noticias actualizadas de Sudamérica
- **API propia (PythonAnywhere)** — API desarrollada y desplegada para gestionar el sistema de likes y dislikes de cada noticia

## Detalles del Proyecto

- **Fecha de desarrollo:** Marzo 2025
- **Duración:** 1 mes
- **Plataforma:** Android (APK)

## Habilidades Demostradas

- Consumo e integración de APIs REST externas
- Desarrollo y despliegue de una API propia con PythonAnywhere
- Desarrollo de aplicaciones móviles Android
- Manejo de almacenamiento local en dispositivos Android
- Manejo y procesamiento de datos en tiempo real
